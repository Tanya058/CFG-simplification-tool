# CFG Simplification Tool

This project is a web-based application that simplifies Context-Free Grammars (CFG) step by step. It helps users understand the process of grammar simplification in an interactive and easy way.

## Features
- Removal of epsilon (ε) productions  
- Removal of unit productions  
- Removal of useless symbols  
- Step-by-step simplification output  
- Simple and user-friendly interface  

## How It Works
The tool simplifies the given grammar in three main steps:
1. Removes null (ε) productions  
2. Eliminates unit productions  
3. Removes non-generating and unreachable symbols  

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## How to Run
1. Download or clone the repository  
2. Open `index.html` in any web browser  
3. Enter CFG rules and click simplify
