📚 CFG Simplification Guide

1. Null (ε) Removal

Eliminates ε-productions while generating equivalent rules to preserve the language.

2. Unit Removal

Removes productions like A → B by replacing them with actual productions of B.

3. Useless Symbol Removal

Removes symbols that cannot generate strings or are not reachable from the start symbol.

⚠️ Order Matters: Null → Unit → Useless
💡 Note: If the start symbol derives ε, a new start symbol may be introduced in advanced simplification.
